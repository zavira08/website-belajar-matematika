Sistem Persamaan Linear Dua Variabel
(SPLDV)

A. Definisi SPLDV

Persamaan linear dua variabel adalah persamaan yang dapat dinyatakan dalam bentuk ax + by = c, dengan x dan y sebagai variabel, serta a,b,c ϵ R (a≠0,b≠0). Apabila terdapat dua buah persamaan linear dua variabel yang berbentuk ax + by = c dan dx + ey = f, dengan persamaan satu dan lainnya saling berkaitan maka kedua persamaan tersebut dinamakan sistem persamaan linear dua variabel. Bentuk umum sistem persamaan linear dua variabel adalah sebagai berikut:


x, y adalah variabel dengan a,b,c,d,e,f ϵ R

Penyelesaian sistem persamaan linear dua variabel yang berbentuk dapat ditentukan dengan mencari nilai x dan y yang memenuhi kedua persamaan linear tersebut. Penyelesaian sistem persamaan linear dua variabel tersebut dapat ditulis sebagai pasangan bilangan berurutan (x, y).

B. Metode Penyelesaian SPLDV

Terdapat beberapa metode yang dapat digunakan untuk menyelesaikan SPLDV, yaitu:

1. Metode Grafik

Penyelesaian dari sistem persamaan linear dua variabel dapat ditentukan dengan menggunakan metode grafik. Langkah-langkah penyelesaian sistem persamaan linear dua variabel dengan menggunakan metode grafik adalah sebagai berikut.

  • Menggambar grafik kedua persamaan pada satu bidang koordinat.
  • Menentukan perkiraan titik potong kedua grafik (jika ada).
  • Memeriksa kembali titik potong kedua grafik dengan memasukkan nilai ke dalam setiap persamaan.

Agar lebih memahami cara penyelesaian sistem persamaan linear dua variabel dengan metode grafik, perhatikan contoh soal berikut.

Contoh Soal:
Tentukan penyelesaian dari sistem persamaan linear dengan menggunakan metode grafik!

Penyelesaian:

Untuk memudahkan kalian dalam menggambar grafik, tentukan titik potong sumbu x dan sumbu y.

2. Metode Substitusi

Penyelesaian dari sistem persamaan linear dua variabel dapat ditentukan dengan menggunakan metode substitusi. Langkah-langkah penyelesaian sistem persamaan linear dua variabel dengan menggunakan metode substitusi adalah sebagai berikut.

  • Pilih salah satu persamaan, misal persamaan I kemudian nyatakan salah satu variabelnya dalam bentuk variabel lain.
  • Substitusikan atau gantikan persamaan yang diperoleh dari langkah pertama ke persamaan lain pada sistem persamaan linear dua variabel.
  • Substitusi nilai variabel yang sudah diperoleh dari langkah kedua pada salah satu persamaan, untuk memperoleh nilai variabel lain.

3. Metode Eliminasi

Penyelesaian dari sistem persamaan linear dua variabel dapat ditentukan dengan menggunakan metode eliminasi. Metode eliminasi adalah mengeliminasi atau menghilangkan satu variabel untuk menyelesaikan variabel lainnya. Langkah-langkah penyelesaian sistem persamaan linear dua variabel dengan menggunakan metode eliminasi adalah sebagai berikut.

  • Eliminasi salah satu variabel dengan menjumlahkan atau mengurangkan kedua persamaan.
  • Hitung nilai variabel lainnya.

Agar lebih memahami cara penyelesaian sistem persamaan linear dua variabel dengan metode eliminasi, perhatikan contoh soal berikut.

Contoh Soal:
Tentukan penyelesaian dari sistem persamaan linear dengan menggunakan metode eliminasi!

Penyelesaian:
1) Untuk mengeliminasi variabel x, persamaan I dikalikan dengan 3 dan persamaan II dikalikan dengan 1.

2) Untuk mengeliminasi variabel y, persamaan I dikalikan dengan 2 dan persamaan II dikalikan dengan 3.

Dari uraian di atas diperoleh nilai x = 5 dan y = 10. Jadi, penyelesaian dari sistem persamaan linear dua variabel adalah (5, 10).

4. Metode Campuran

Penyelesaian dari sistem persamaan linear dua variabel dapat ditentukan dengan menggunakan metode campuran. Metode campuran adalah gabungan dari metode substitusi dan eliminasi. Langkah-langkah penyelesaian sistem persamaan linear dua variabel dengan menggunakan metode campuran adalah sebagai berikut.

  • Eliminasi salah satu variabel.
  • Substitusikan nilai variabel ke salah satu persamaan pada sistem persamaan linear dua variabel.

Agar lebih memahami cara penyelesaian sistem persamaan linear dua variabel dengan metode campuran, perhatikan contoh soal berikut.

Contoh Soal:
Tentukan penyelesaian dari sistem persamaan linear dengan menggunakan metode campuran!

Penyelesaian:
1) Eliminasi salah satu variabel. Untuk mengeliminasi variabel x, persamaan I dikalikan dengan 1 dan persamaan II dikalikan dengan 2.

2) Substitusikan nilai variabel ke salah satu persamaan pada sistem persamaan linear dua variabel. Substitusi nilai y=5 ke persamaan I.

Dari uraian di atas diperoleh nilai x = -4 dan y = 5. Jadi, penyelesaian dari sistem persamaan linear dua variabel adalah (-4, 5).
